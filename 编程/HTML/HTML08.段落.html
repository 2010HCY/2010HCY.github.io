HTML 段落

HTML 可以将文档分割为若干段落。

HTML 段落

段落是通过 <p> 标签定义的。

<p>这是一个段落 </p>
<p>这是另一个段落</p>

效果：

这是一个段落

这是另一个段落

注意哈：浏览器会自动地在段落的前后添加空行。（</p> 是块级元素）

不要忘记结束标签

虽然像这样忘了使用结束标签，大多数浏览器也会正确地将 HTML 显示出来：

<p>这是一个段落
<p>这是另一个段落

上面的例子在大多数浏览器中都没问题，但不要这么做，忘记使用结束标签会产生意想不到的结果和错误，而且万一出啥问题你一个个改得累死

注: 在未来的 HTML 版本中，不允许省略结束标签。

HTML 换行

如果你希望在不创建一个新段落的情况下进行换行（新行），请使用 <br> 标签，就像这样：

<p>门罗币钱包地址<br>47JGXgtfPvANa9b29RfGEGRpCsK6<br>2MtffCYvxfcpYBHpikUAxqTXLHs9o<br>XHT4bkszEL9HuDzw5mtkjDk4o2Kx<br>KQmRM6G4vv</p>

效果

门罗币钱包地址
47JGXgtfPvANa9b29RfGEGRpCsK6
2MtffCYvxfcpYBHpikUAxqTXLHs9o
XHT4bkszEL9HuDzw5mtkjDk4o2Kx
KQmRM6G4vv

<br /> 元素是一个无意义 HTML 元素。由于换行标签不需要结束标签，因此它没有结束标签。

HTML 输出- 使用提醒

我们在编辑时无法确定 HTML 页面显示的确切效果。屏幕的大小，以及对窗口的调整都可能导致不同的结果。同时，对于 HTML，你无法通过在 HTML 代码中用的空格或换行来改变输出的文字位置。

当显示页面时，浏览器会移除源代码中多余的空格和空行。所有连续的空格或空行都会被算作一个空格。需要注意的是，HTML 代码中的所有连续的空行（换行）也被显示为一个空格。

比如:

<p>1<br><br><br>2                  3</p>

效果:

1


2 3
